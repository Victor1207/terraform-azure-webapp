\# Terraform Azure Web App Project



\## Overview

This project demonstrates a full end-to-end Azure deployment using Terraform. It includes:



\- Azure Resource Group, App Service Plan, and Windows Web App

\- Observability with Log Analytics and Application Insights

\- Autoscaling based on CPU metrics

\- Availability testing with Application Insights web tests

\- Alerts using Azure Monitor and Action Groups

\- Policy-compliant infrastructure (Environment tags)

\- Zero manual configuration (100% Infrastructure as Code)



This project is designed to showcase \*\*DevOps/SRE practices\*\* for cloud infrastructure.



---



\## Tech Stack

\- \*\*Cloud Provider:\*\* Microsoft Azure

\- \*\*IaC Tool:\*\* Terraform

\- \*\*Observability:\*\* Application Insights, Log Analytics, Azure Monitor

\- \*\*Language:\*\* HCL (Terraform)

\- \*\*OS:\*\* Windows (App Service Plan)

\- \*\*CI/CD Ready:\*\* Fully compatible with GitHub Actions or Azure DevOps
